# Web3 Content Factory

AI 驱动的 Web3 内容增长引擎 —— 你不是在手动发帖，你在训练一个增长模型。

## 三大核心模块

### 模块一：热点即插即用（Newsjack Engine）
- LLM + RAG 实时抓取市场热点
- 热点到内容生成到多版本 A/B 测试的自动化 SOP
- 一个 Brief 输出 100+ 条不同风格推文

### 模块二：内容形态工厂（Content Factory）
输入一个交易活动 Brief，自动输出：
- 5 个不同角度的推文 Hook
- 信息图的数据脚本
- 短视频的逐帧脚本
- Spaces 主持人提问话术

### 模块三：自动化分发与迭代（Auto-Distribution）
- 合规多账号/多时区自动排期分发
- 数据反馈驱动 Prompt 自动优化
- 持续迭代的增长模型

## 标准作业流程 SOP

Brief 输入 → 热点匹配 → 批量生成 → 质检筛选 → 排期分发 → 数据回流 → 自动优化
